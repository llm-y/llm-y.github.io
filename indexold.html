Pertanyaan Umum

Apakah perlu instalasi Node/NPM?
Tidak. Halaman ini murni HTML + CSS + Vanilla JS (ES Modules). Integrasi API dapat ditambahkan via fetch() jika backend siap.
Bagaimana cara kerja tombol Why??
Di aplikasi penuh, Why? menampilkan alasan terstruktur (faktor kunci, asumsi, alternatif, dan sitasi) untuk setiap jawaban. Di demo ini, konten penjelasan bersifat contoh statis.
Bisakah menghasilkan sitasi otomatis?
Ya. Mode sitasi menambah rujukan pada klaim penting. Implementasi produksi akan menautkan DOI/URL primer.
